# Simple Interest Calculator

## Description
Ce projet calcule l'intérêt simple basé sur le principal, le taux et le temps.

## Formule
Intérêt Simple = (Principal × Taux × Temps) / 100

## Utilisation
```bash
./simple-interest.sh
```
